## Projets Node.js

#### Description
Ce dépôt comporte trois projets effectués dans un cadre scolaire. Tous comportent un côté serveur et un côté client. Le premier est très simple. Les deux autres sont transactionnels et sont beaucoup plus complexes, se basant sur un cours d'OpenClassrooms intitulé: *Passez au Full Stack avec Node.js, Express et MongoDB* au lien ci-dessous:

Lien: https://openclassrooms.com/fr/courses/6390246-passez-au-full-stack-avec-node-js-express-et-mongodb

Le produit final est un site web transactionnel permettant d'ajouter des produits, modifier leur contenu ou les supprimer. Chaque produit comporte quelque champs tel un nom, une description, une photo ainsi qu'un prix.

#### Instructions
1) À partir du répertoire "frontend", executer dans une console la commande *npm run start*
2) À partir du répertoire "backend", dans une autre console, executer la commande *node server.js*
3) Utiliser l'addresse *localhost:4200* dans le navigateur
